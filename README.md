# apod
Desktop viewer for astronomy picture of the day

Get free API key: https://api.nasa.gov/

### Version
1.0.0

## Usage

### Installation

Install the dependencies
```sh
$ npm install
```

Create .env file
```sh
$ touch .env
```
Set API_KEY variable in .env
```sh
API_KEY=xxxxxxxxxxxxxxxxxxxxx
```

### Serve
To run electron

```sh
$ npm start
```
